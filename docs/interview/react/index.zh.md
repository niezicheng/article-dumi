### Link 标签与 a 标签？

**react-router 里面的 Link 标签和 a 标签的区别？**

> 本质渲染到页面上展示的都是 `a` 标签

- `Link` 中禁用了 `a` 标签的默认事件
- `Link` 改用了 `history` 对象提供的方法进行跳转
- `Link` 的 `跳转` 行为只会触发相匹配的 `Route` 对应的页面内容更新，而不会刷新整个页面
